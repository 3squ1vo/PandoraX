## 💭 🛠️ The Philosophy and Engineering behind the madne... Pandora (By Luiz Carlos)

Pandora is something basic, yet way out of the curve; she is Pandora and I created her just to help me with schedules and also to make me laugh even when I'm coding sad or angry. The biggest challenge was what every programmer suffers from: making something, getting an error, and saying *"What on earth is making this crash?"*.

I'm publishing it right now because a lot of people are looking for an A.I. (for... *ahem*, weird reasons). Many want to turn it into their "virtual girlfriend", while others just want someone to help them with whatever they are doing, besides having someone to talk about their feelings.

A huge recommendation for whoever uses her: if you use her and/or record a video with her, feel free to do so. I accept criticism (as long as it's constructive) and compliments about her. Also, if you don't like how she is right now, you have the freedom to modify her to your liking. One thing I really want is for you to code and use her. I'm giving you the skeleton/draft of something with a good or even giant potential. Just code, because programming is a very relaxing thing, and when we code, or even when something we make finally works, the feeling is incredibly rewarding.

And that's the feeling I want to pass on when you use her and/or make any changes. It won't be easy, but nothing a little help can't fix. I won't deny I used a little help myself, if you know what I mean... But it's better to ask for help knowing what you're doing than not doing it at all.

Another thing: for those who think they are "badass devs", you're just going to get stressed, lose interest, and criticize without context or just to boost your bruised ego (in case you know Python, which is her language). And for those who don't know how to code: patience. The important thing is not giving up at the first hurdle; Pandora is a test of perseverance with a great reward. So, if you're going to use an A.I. to help you add a feature you want? Wonderful! Regardless of whether you know the language or not, just be curious and want to learn or know what you are doing. Because if someone asks "how did you do that?", you can answer with conviction and know what you're talking about. This goes for the `ctrl + c` and `ctrl + v` crowd too. Because when an error pops up, even if you paste it into the chat, the A.I. might hinder more than help. Remember: it doesn't know everything.

**And she is here to help you, not to be used as a crutch.**

## 🚀 Pandora's Features

* **Cross-OS Automation:** Web navigation, system control, and a deep scanning engine (`os.walk`) capable of dynamically finding and opening folders.
* **Voice Synthesis:** Integration with Edge-TTS and an intelligent Cache system (MD5 Hash) for repeated requests.
* **Anti-Echo System:** Yeah, a system to prevent Pandora's echo. Pandora (at least on my OS) used to speak twice, so I added an anti-echo system.

### Prerequisites (Linux)
```bash
sudo apt update
sudo apt install portaudio19-dev python3-pyaudio git
```

Note: The prerequisites above were the ones I needed to make Pandora work for me, but your mileage may vary.

Running the Project
```
git clone [https://github.com/SEU_USUARIO/PandoraX.git](https://github.com/SEU_USUARIO/PandoraX.git)
cd PandoraX
pip install -r requiements.txt
```
