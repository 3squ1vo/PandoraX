## 💭 🛠️ A Filosofia e Engenharia por trás da loucu... Pandora (Por Luiz Carlos)

A Pandora é algo básico, mas bem fora da curva; ela é a Pandora e a criei só para me ajudar com agendas e também para me fazer rir até quando estou programando triste ou com raiva. O maior desafio foi o que todo programador sofre, que é tu fazer algo e dar um erro ao ponto de você falar *"Mas o que está fazendo isso dar erro?"*.

No momento estou publicando, pois tem muita gente que está atrás *(lá ele)* de uma I.A. Muitos são para transformar em sua "namorada virtual", já outros é para ter alguém para ajudar no que esteja fazendo, além de poder falar o que sente. 

Uma grande recomendação para quem for usar ela é que, se for usar e/ou gravar algum vídeo com ela, fique à vontade para usá-la. Aceito críticas (desde que sejam construtivas) e elogios sobre ela. Também, caso não goste de como ela está, tem liberdade de poder modificar ela para ficar ao seu gosto. Uma coisa que eu quero é que você programe e a use. Eu estou dando o esqueleto/esboço de algo com um bom ou até um gigante potencial. Só programe, pois a programação é algo bem relaxante e quando programamos, ou até quando algo que fazemos dá certo, a sensação é recompensadora demais.

E essa é a sensação que quero passar quando for usar ela e/ou fazer alguma alteração. Não será fácil, mas nada que uma ajudinha não possa resolver. Eu mesmo não nego que usei uma ajudinha, se é que você me entende... Mas é melhor pedir uma ajuda sabendo o que está fazendo do que não fazer.

Outra coisa: para quem se acha o "fodão", só vai passar raiva, vai dar desinteresse e vai criticar sem nexo ou para melhorar o seu ego ferido (caso saiba programar em Python, que é a linguagem dela), e para quem não sabe programar, paciência. O importante é não desistir na primeira dificuldade, a Pandora é um teste de perseverança com uma boa recompensa. Então, se for usar uma I.A. para lhe ajudar a adicionar algo que você quer?? Maravilhoso! Pois independentemente se sabe da linguagem ou não, só seja curioso e queira aprender ou saber o que está fazendo. Pois se alguém perguntar como tu fez isso e tals, você poder responder com convicção e sabendo o que está falando. Isso também é para o pessoal do `ctrl + c` e `ctrl + v`. Pois, quando der um erro e mesmo que você coloque no chat, a I.A. pode lhe atrapalhar mais do que ajudar. Se lembre: ela não sabe de tudo.

**E ela é para lhe ajudar, não para ser usada como muleta.**

## 🚀 As funcionalidades da Pandora

* **Automação Cross-OS:** Navegação web, controle de sistema e motor de varredura profunda (`os.walk`) capaz de encontrar e abrir pastas dinamicamente.
* **Síntese de Voz:** Integração com o Edge-TTS e sistema inteligente de Cache (MD5 Hash) para requisições repetidas.
* **Sistema Anti-Eco:** É, um sistema para evitar o eco da Pandora, pois a Pandora (pelo menos no meu SO) falava duas vezes com isso adicionei um sistema anti-eco.

### Pré-requisitos (Linux)
```bash
sudo apt update
sudo apt install portaudio19-dev python3-pyaudio git
```

Obs: Os pré-requisitos acima foram que eu precisei usar para a Pandora funcionar comigo, mas são casos e casos.

Rodando o Projeto
```
git clone [https://github.com/SEU_USUARIO/PandoraX.git](https://github.com/SEU_USUARIO/PandoraX.git)
cd PandoraX
pip install -r requiements.txt
```
